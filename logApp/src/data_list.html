1.   Introduction

The telescope performance monitor (TPM) monitors many telescope parameters during a night's observation. These parameters are dumped into a log file on APO's Unix box. To make it easier for operators to refer to specific parameters, names have been designated for each data point. This document describes the name-to-parameter mapping.

These tables are a work-in-progress. As new instrumentation gets added (or removed), these tables will be updated appropriately.

The tables use a color coding to reflect the state of support. The colors are defined to be:

Color Description
  Has TPM support and is enabled by default.
  Has TPM support but isn't enabled by default.
  No TPM support yet.
