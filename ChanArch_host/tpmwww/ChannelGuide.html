SDSS Telescope Performance Monitor Channel Guide

This page contains links to a number of different views, or reports , of the TPM EPICS database contents. These reports use the system and subsystem designations defined in the Systems and Subsystems page.

Users can also obtain information on specific channels via a query interface.

Get Channel Information

Please enter the name of a TPM channel:
The following table provides a list of fields that will appear in the reports. Of these, the first five (NAME, ARCHIVED, SYSTEM, SUBSYSTEM, and DESC) are of general interest. The remainder describe some aspects of the EPICS database configuration.

Field Description
NAME EPICS database record name
ARCHIVED Is this channel archived?
N - NO
VAL - default record value archived
VALA-Z - fields VALA through VALZ of record archived
SYSTEM System name
SUBSYSTEM Subsystem name
DESC Description of record contents
RTYP EPICS database record type
SCAN EPICS database record scanning mechanism
DTYP EPICS database device support type
INP EPICS database input link specifier
OUT EPICS database output link specifier
