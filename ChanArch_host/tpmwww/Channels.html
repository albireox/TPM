SDSS Telescope Performance Monitor Channels

The TPM channels archived by the EPICS Channel Archiver are grouped into the six configuration files described in the following table.

A note on syntax: For groups of channels with similar names I will often use the following two constructs.

  • [a..b] - Follow the ASCII character sequence from 'a' to 'b'. For example, [0..4] generates five names using '0', '1', '2', '3, '4'.
  • {ab,bc,..,yz} - Select each of the listed choices. For example, [AL,AZ,RO] generates three names using 'AL', 'AZ', 'RO'.

    • Each of these are independent, thus a name pattern using both [0..4] and {AL,AZ,RO} will generate 15 unique names.

    Configuration File Contents
    ab.cfg Allen-Bradley data
    admin.cfg TPM administrative data
    ampslip.cfg Amplifier status and slip detection data
    interlocks.cfg Selected interlock system status words
    mirrors.cfg M1 and M2 MIG and Galil data
    mount.cfg Mount motor and encoder data
    pmss.cfg PMSS data
    temperatures.cfg Temperature sensor data

    tpm_ABO2_[00..31] 32 16-bit words from file O2 (Output)
    tpm_ABI3_[00..07] 8 16-bit words from file I3 (Input)
    tpm_ABI4_[00..07] 8 16-bit words from file I4 (Input)
    tpm_ABI5_[00..07] 8 16-bit words from file I5 (Input)
    tpm_ABI6_[00..01] 2 16-bit words from file I6 (Input)
    tpm_ABI7_[00..01] 2 16-bit words from file I7 (Input)
    tpm_ABI8_[00..01] 2 16-bit words from file I8 (Input)
    tpm_ABI9_[00..01] 2 16-bit words from file I9 (Input)
    tpm_ABI10_[00..01] 2 16-bit words from file I10 (Input)
    tpm_ABO11_[00..01] 2 16-bit words from file O11 (Output)
    tpm_ABO12_[00..01] 2 16-bit words from file O12 (Output)

    admin.cfg
    Channel Name Description
    tpm_uptime TPM uptime in hours
    tpm_MJD SDSS Modified Julian Date

    ampslip.cfg
    Channel Name Description
    tpm_{AL,AZ}[1..2]_Ampstat Altitude and Azimuth motor amplifier status words
    tpm_ROT_Ampstat Rotator motor amplifier status word
    tpm_{AL,AZ}[1..2]_Slip Altitude and Azimuth motor slip detection status values
    tpm_ROT_Slip Rotator motor slip detection status value

    interlocks.cfg
    Channel Name Description
    tpm_{AL,AZ,RO}STATE Axis state. Meaning?
    tpm_M2FORCE M2 Force Limit Switch!
    tpm_{AL,AZ,RO}BRK{EN,DIS}CMD Axis brake enable or disable command
    tpm_{AL,AZ,RO}BRK{EN,DIS}STAT Axis brake enable or disable status
    tpm_{AL,AZ,RO}BRK{EN,DIS}MCP MCP's idea of the axis brake
    enable or disable status

    mirrors.cfg
    Channel Name Description
    tpm_MIG1{A,B,C,DT,EL,FL} M1 MIG readings: axials A,B,C,
    transverse D, laterals E,F
    tpm_MIG2{A,B,C,DT} M2 MIG readings: axials A,B,C,
    transverse D
    tpm_M1GAL[0..5]A M1 Galil actual positions
    tpm_M1GAL[0..5]C M1 Galil commanded positions
    tpm_M1GAL[0..5]H M1 Galil homed status
    tpm_M2GAL[0..4]A M2 Galil actual positions
    tpm_M2GAL[0..4]C M2 Galil commanded positions
    tpm_M2GAL[0..4]H M2 Galil homed status

    mount.cfg
    Channel Name Description
    tpm_{AL,AZ,RO}CPOS Axis commanded position
    tpm_{AL,AZ,RO}CVOLT Axis motor amplifier commanded voltage
    tpm_{AL,AZ}ENC[1..2] Altitude and Azimuth encoder readings
    tpm_ROCENC Rotator encoder reading
    (Should be named tpm_ROENC)
    tpm_{AL,AZ}MTRC[1..2] Altitude and Azimuth motor currents
    tpm_ROMTRC Rotator motor current
    tpm_{AL,AZ}MTRV[1..2] Altitude and Azimuth motor voltages
    tpm_ROMTRV Rotator motor voltage
    tpm_{AL,AZ,RO}ERR Axis servo errors from the MEI
    tpm_{AL,AZ,RO}MCP{POS,VEL,TIM} Axis MCP PVT triplets.
    (Obtained from the MEI in a undocumented fashion)
    tpm_{AL,AZ,RO}TCC{POS,VEL,TIM} Axis TCC PVT triplets.
    (Interpretation?)

    pmss.cfg
    Channel Name Description
    tpm_PMSS.VAL[A..D] Position, position error, pressure,
    and pressure error measured in volts
    for PMSS Zone 1: Axial A.
    tpm_PMSS.VAL[E..H] Position, position error, pressure,
    and pressure error measured in volts
    for PMSS Zone 2: Axial B.
    tpm_PMSS.VAL[I..L] Position, position error, pressure,
    and pressure error measured in volts
    for PMSS Zone 3: Axial C.
    tpm_PMSS.VAL[M..P] Position, position error, pressure,
    and pressure error measured in volts
    for PMSS Zone 4: Transverse D.

    temperatures.cfg

    Channel Name Description
    tpm_TM_M0B[0..7]Chans.VAL[A..H] M1 (module 0) temperature sensors
    on blocks 0,1,2,3,4,5,6,7
    VAL[A..H] reference the eight channels
    on each block.
    tpm_TM_M1B[0..7]Chans.VAL[A..H] M2 and Truss (module 1) temperature sensors
    on blocks 0,1,2,3,4,5,6,7.
    VAL[A..H] reference the eight channels
    on each block.
    tpm_TM_M2B[0..7]Chans.VAL[A..H] Telescope (module 2) temperature sensors
    on blocks 0,1,2,3,4,5,6,7.
    VAL[A..H] reference the eight channels
    on each block.
    tpm_TM_M[0..2]B[0..7]Chans.VAL[I..P] VAL[I..P] for all of the above records
    are not archived but are used to drive the
    temperature graphics. Their values are the
    relative differences between the
    corresponding VAL[A..H] and the ambient
    scaled by tpm_TM_Scale.
