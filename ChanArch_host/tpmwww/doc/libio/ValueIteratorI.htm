ValueIteratorI Class

Value iterator iterface to be implemented. The preferred API is ValueIterator.

Interface

  • Virtuals to be implemented by derived classes: 
    virtual bool isValid () const = 0;
virtual const ValueI * getValue () const = 0;
virtual bool next () = 0;
virtual bool prev () = 0;
  • Number of "similar" values that can be read as one chunk in a block operation. (This call might go away when a MemoryArchive & network request mechanism is implemented. Let me know if you depend on this call. The result might also not be perfect, e.g. a single 8000 chunk could be reported as two 4000-sized chunks etc.) 
    virtual size_t determineChunk (const osiTime &until) = 0;
  • Sampling period for current values in secs. 
    virtual double getPeriod () const = 0;
