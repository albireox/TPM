Command: dump.tcl


Syntax: tcl dump.tcl archive channelName [ startTime [ endTime ] ]
              start/endTime as "YYYY/MM/DD hh:mm:ss" in 24h format

Function:

Dump values for given channel, either all values or between start and end time.

Example Output:

kasemir@ioc94 atac> tcl dump.tcl $ARCH ExampleInfo "2000/02/17 18:03:00"
2000/02/17 18:03:00.042515200 Test Run Nr. 3027
2000/02/17 18:03:00.092587200 Test Run Nr. 3028
2000/02/17 18:03:00.142659200 Test Run Nr. 3029
.....
