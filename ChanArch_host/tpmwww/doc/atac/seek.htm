Command: seek.tcl


Syntax: tcl seek.tcl archive channelName time
              time as "YYYY/MM/DD hh:mm:ss" in 24h format

Function:

For given channel, show values before/near/after time.

Example Output:

kasemir@ioc94 atac> tcl seek.tcl $ARCH ExampleInfo "2000/02/17 18:03:00.09"
before: 2000/02/17 18:03:00.042515200 Test Run Nr. 3027
near: 2000/02/17 18:03:00.092587200 Test Run Nr. 3028
after: 2000/02/17 18:03:00.092587200 Test Run Nr. 3028
