ABDUMP - Dump the Interlock Data

To create a interlock file in survey standard Yanny parameter file format:

[sdsshost:]$ setup tpm
[sdsshost:]$ abdump

Usage: abdump outfile {-cnt {cnt 1}} month day hms {year THISYEAR}

outfile = name of Yanny parameter file to create
cnt = Number of 1Hz samples to acquire
month = Jan,Feb,Mar,Apr,May,Jun,Jul,Aug,Sep,Oct,Nov,Dec
day = 1..31
hms = HH:MM:SS (must be in UT)
year = Gregorian year, e.g. 2002
The default year selects between 1 Jan and TODAY of this year and between TODAY and 31 Dec of last year.

To create the interlocks graphical display using IOP:

When accessing a multi-sample file, i.e. if you specified cnt to be greater than 1 in your adbump call, you have to specify the sample number from 0 to N-1, e.g.

[sdsshost:]$ setup iop
[sdsshost:]$ iop
im> startInterlocks -f tpm.par:2

abdump is implemented using the casiwish (Tcl) scripting interface to the Channel Archiver files.
