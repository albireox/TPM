Command: PlotBrowser.tk

Syntax: wish PlotBrowser.tk archive channelName ?channelName? ...

Function:

Browse values for channels in a plot.

Example Snapshot:

The intend is not to provide a complete plotting tool but a simple example as a basis for specialized tools.
It indicates "informational" values (channel disconnected, Archiver off, ...) as shown in the example snapshot with an 'x'.
This program handles only scalar values. It does not catch all possible errors like no data available etc.
The PlotBrowser is not optimized, each pan or zoom operation causes the program to re-read the specified time range.
