Command: ListBrowser.tk


Syntax: wish ListBrowser.tk archive channelName [ endTime ]
              endTime as "YYYY/MM/DD hh:mm:ss" in 24h format

Function:

Browse values for given channel in a listbox.

Example Snapshot:

The original motivation for this tool was to allow browsing of a "string" channel that was used to log operator comments.
Whenever an interesting entry is found, the associated time stamp can then be copied-and-pasted into another tool to look at the numerical data for that time regime.
