ChannelIterator Class

Iterator for looping over channels, can be used similar to ChannelI pointer:

Examples

  • Archive archive (new BinArchive (archive_name));
ChannelIterator channel (archive);

    
archive.findFirstChannel (channel);
while (channel)
{
	cout << channel->getName() << endl;
	++ channel;
}

(Implemented as "smart pointer" wrapper for ChannelIteratorI interface)

Interface

  • Obtain new, empty ChannelIterator suitable for given Archive 
    ChannelIterator (const class Archive &archive);
  • Does Iterator hold valid channel ? 
    operator bool () const;
  • Access to Channel 
    ChannelI *operator -> ();
ChannelI &operator * ();
  • Move to next channel 
    ChannelIterator &operator ++ ();
